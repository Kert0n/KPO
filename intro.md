# Введение

Этот конспект посвящен конструированию программного обеспечения как практической дисциплине: как принимать проектные решения, удерживать сложность, строить границы модулей и писать код, который можно развивать.

## Как читать

Каждая лекция устроена одинаково:

- цель;
- ключевые идеи;
- основной конспект;
- пример;
- вопросы для самопроверки;
- материалы для продолжения.

## Обозначения

Примеры кода даны на Kotlin, C#, Java и Go. Если пример иллюстрирует одну идею сразу на нескольких языках, он оформляется единым блоком с переключением языка.
