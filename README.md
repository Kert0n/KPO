# Конспект КПО

VitePress-проект для русскоязычного конспекта по конструированию программного обеспечения.

## Команды

```bash
npm install
npm run dev
npm run build
npm run preview
```

Для локальной разработки откройте адрес, который покажет `npm run dev`.

## Структура

- `intro.md` - вводная секция.
- `lectures/*.md` - лекции. Файлы подхватываются автоматически.
- `extras/*.md` - дополнительные материалы. `extras/index.md` является обзорной страницей и не попадает в список дополнительных материалов.
- `conclusion.md` - заключение.

## Автонавигация

Лекции и дополнительные материалы не нужно добавлять в `.vitepress/config.mts`.

Порядок страниц:

1. `order` во frontmatter.
2. Числовой префикс имени файла, например `01-`.
3. Алфавитный порядок.

Название страницы:

1. `title` во frontmatter.
2. Первый заголовок `#`.
3. Имя файла, преобразованное в текст.

## Многоязычные примеры

Используйте контейнер `multi-code`:

````md
::: multi-code "Обработка команд приложения" {default=kotlin playground=kotlin}

```kotlin
fun main() {
    println("Kotlin")
}
```

```csharp
Console.WriteLine("C#");
```

```java
System.out.println("Java");
```

```go
fmt.Println("Go")
```

:::
````

Поддерживаемые языки: Kotlin, C#, Java, Go. Если в `multi-code` есть Kotlin-блок, он автоматически пробует открыться через JetBrains Kotlin Playground и откатывается к обычной подсветке, если Playground недоступен.

Важно: при запуске Kotlin Playground код отправляется в инфраструктуру Playground для компиляции и выполнения.

Верхняя панель содержит глобальный тумблер `Playground`. Он выключает интерактивные Kotlin-редакторы на всех страницах и сохраняет выбор в браузере. Тема Playground синхронизируется с темой сайта: `idea` для светлой темы и `darcula` для темной.
