# Дополнение 01

_Шаблон дополнительного раздела. Переименуйте заголовок — он автоматически попадёт в боковую панель._

## Содержание

Текст дополнения. Здесь работают те же возможности, что и в лекциях, включая переключалку языков:

::: multi-code "Пример в дополнении" {default=go}

```kotlin
fun greet(name: String) = "Привет, $name!"
```

```go
package main

func greet(name string) string {
    return "Привет, " + name + "!"
}
```

:::

Здесь язык по умолчанию — Go (опция `{default=go}`), а перевод есть только на два языка.
