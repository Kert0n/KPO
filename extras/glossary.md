# Глоссарий

## Архитектурная граница

Место, где одна часть системы перестает знать детали реализации другой части и взаимодействует с ней через договоренность: интерфейс, протокол, событие или формат данных.

## Модуль

Единица организации кода с собственной ответственностью, публичным API и скрытыми деталями реализации.
