# Проверки

`npm run check` выполняет lint, format check, TypeScript checks, content contract и unit tests.

`npm run verify` дополнительно собирает сайт, проверяет performance budget, компилирует runnable Kotlin examples и запускает browser tests.

Для Kotlin compilation нужен JDK 21. CI предоставляет его автоматически.
