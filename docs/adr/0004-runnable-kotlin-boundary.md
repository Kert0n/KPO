# ADR 0004: граница runnable Kotlin

Статус: принято.

Технически проверяются только fences `kotlin playground`: они уже заявляют выполнение в браузере. Обычные иллюстративные фрагменты и блоки с `{playground=off}` не компилируются.

Компиляция Kotlin подтверждает синтаксическую и типовую целостность runnable-примера, но не оценивает знания студента, учебный смысл или эквивалентность примеров на других языках.
