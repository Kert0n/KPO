# ADR 0002: расширение default theme VitePress

Статус: принято.

KPO остаётся статическим сайтом на VitePress и расширяет его стандартную тему. Это сохраняет совместимость с local search, навигацией, GitHub Pages и существующим Markdown DSL без собственной платформы или backend.

VitePress-specific CSS остаётся адаптерным слоем; продуктовая логика не должна зависеть от его DOM за пределами интеграции темы.
