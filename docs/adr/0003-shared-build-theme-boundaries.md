# ADR 0003: границы shared, build и theme

Статус: принято.

Чистые модели, идентификаторы блоков, хеширование и правила контента находятся в `shared`. Build-time плагины и скрипты используют Node и каталог. Vue-компоненты и DOM-логика остаются в `theme`.

Такой разрез позволяет проверять общую логику unit-тестами и не тянуть Vue/DOM в генерацию сайта.
